# IDE Integration

## VS Code (Continue.dev)
1. Install **Continue** extension.
2. Select **Ollama** as the provider.
3. Config (`config.json`):
   - Model: `qwen2.5-coder:7b`
   - Tab Autocomplete: Enable.

## Neovim (Ollama.nvim)
1. Use `ollama.nvim` or `gen.nvim`.
2. Map `<leader>ai` to trigger local LLM prompts.
